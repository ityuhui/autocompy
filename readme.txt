1.修改defaults\preferences\defaults.js
填入ibm internat用户名和密码

2.在autocompy目录下，选中chrome文件夹，defaults文件夹，chrome.manifest,install.rdf，将其压缩为一个zip文件，如autocompy.zip，将其改名为autocompy.xpi

3.将autocompy.xpi用鼠标拖入firefox中

4.将firefox的add-on栏打开，方法是在firefox的工具栏附近右键鼠标，勾选Add-on Bar

5.如果add-on栏出现在firefox的下端，并且右侧显示 complete it! 说明安装成功。当需要自动填充用户名和密码时，只需要点击complete it!
